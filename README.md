
# Galería San Luis

Sitio web turístico desarrollado con React + Vite + TypeScript que muestra lugares destacados de la provincia de San Luis, Argentina.

## Tecnologías utilizadas

- React
- TypeScript
- Vite
- React Router
- CSS Modules

## Funcionalidades

- Navbar con navegación entre páginas
- Header con slider automático de imágenes
- Secciones de turismo (Potrero, Trapiche, Merlo)
- Sección de videos turísticos
- Página final de invitación a visitar San Luis
- Diseño responsive
